2. Allgemeines

Der Schutz Ihrer personenbezogenen Daten ist mir wichtig. Die App RetroEdit kann vollständig offline genutzt werden und erhebt keine personenbezogenen Daten, es sei denn, dies wird in dieser Erklärung ausdrücklich beschrieben.

3. Erhebung und Verarbeitung personenbezogener Daten

Keine Datenerhebung durch die App selbst:

  • Es werden keine personenbezogenen Daten gesammelt.
  • Es werden keine persönlichen Informationen gespeichert.
  • Es werden keine Benutzerkonten verwendet.
  • Bilder und Inhalte werden ausschließlich lokal verarbeitet.
  • Es werden keine Daten an eigene Server oder an Dritte gesendet (außer Google Analytics, siehe unten).

Die App verwendet Google Analytics zur anonymisierten Auswertung von Nutzungsstatistiken. Google kann dabei Daten wie:

  • Nutzungsverhalten (z. B. Feature-Aufrufe)
  • Geräteinformationen (Modell, OS-Version)
  • Anonymisierte Nutzungsereignisse

Es findet keine Identifizierung einzelner Nutzer statt.

Weitere Informationen: https://policies.google.com/privacy

5. Offline-Nutzung

Die App kann vollständig offline verwendet werden. Ohne Internetverbindung erfolgt keinerlei Datenübertragung.

6. Weitergabe von Daten

Es erfolgt keine Weitergabe personenbezogener Daten an Dritte, außer im Rahmen der Nutzung von Google Analytics.
